About

This is a Simple Web Application which displays graphs of consumers' data.

Heavily relying on the v4.0.0-alpha.6. Bootstrap library, the d3.js library and built using the AngularJS framework.

About Bootstrap

Bootstrap is a library which provides loads of useful components and templates to create visually pleasing and responsive websites.

Some components used to make this app: sticky navbar, jumbotron, card (new v4 component).

About d3.js

d3.js is a library which enables to represent data in a efficient way.

On this app, the data is fetched from JSON files and displayed using a line chart.

About Angular JS

Angular JS is a JavaScript framework using the MVC model which makes the Simple Web Applications creation process easy.
